# Getting Started

## Prerequisites

- **Conda**: Anaconda or Miniconda.
- **Operating System**: macOS (Apple Silicon recommended), Linux, or Windows.

## Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/StartLivin/neolat.git
   cd neolat
   ```

2. **Run the setup script**:

   ```bash
   chmod +x setup_env.sh
   ./setup_env.sh
   ```

3. **Activate the environment**:
   ```bash
   conda activate neolat
   ```

## Quick Start

Run a simple test agent:

```bash
python -m neolat.experiments.run_test
```
